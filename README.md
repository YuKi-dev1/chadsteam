# chadsteam 🏴‍☠️
**O ChadSteam é uma ferramenta feita para instalar facilmente manifests junto com o SteamTools.**

Esta ferramenta foi criada com o íntuito pra ajudar as pessoas que têm dificuldades de instalar ou encontrar boas fontes.
Esta ferramenta tem várias categorias com coisas diferente como:

- Loja (é onde irás instalar os teus jogos/manifest);
- Biblioteca (É onde vais gerenciar os teus jogos);
- Requesitar (Podes requesitar desde jogos e atualizações caso falte DLCS);
- E muito mais...
